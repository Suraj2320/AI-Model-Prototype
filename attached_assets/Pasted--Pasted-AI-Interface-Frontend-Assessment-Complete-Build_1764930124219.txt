# AI Interface Frontend Assessment - Complete Build Prompt

## Project Overview
Build a polished, frontend-only prototype of an AI chat interface (similar to ChatGPT, Claude, Gemini) that showcases modern UI/UX design and React/TypeScript development skills.

## Tech Stack (REQUIRED)
- **Framework**: Next.js OR React.js (functional components only)
- **Language**: TypeScript with strict mode enabled
- **Styling**: Tailwind CSS
- **Deployment**: Netlify, Vercel, or GitHub Pages

### AI Platforms to Review (Choose 3-5)
- OpenAI Playground
- Hugging Face Spaces
- Anthropic Claude UI
- Microsoft Copilot Lab
- Google Gemini
- Perplexity AI
- Any other AI chat interface

### Research Documentation (for README)
For each platform reviewed, write **1-2 sentences** summarizing:
- What makes their interface intuitive
- Key standout features you noticed
- Any unique interaction patterns

### Feature Selection
Choose **4-6 features** from your research to implement. Examples:
- Model/temperature selection
- Conversation history sidebar
- Copy/regenerate response buttons
- Syntax highlighting for code
- Streaming text effect
- Token counter
- Export conversation
- Prompt templates/examples
