## UI/UX Best Practices for Realistic Design

### Modern AI Interface Patterns
1. **Conversation Flow**: Messages alternate between user and assistant with clear visual distinction
2. **Input Area**: Fixed at bottom with auto-expanding textarea
3. **Streaming Effect**: Simulate typing animation for responses (use setInterval to reveal text gradually)
4. **Action Buttons**: Copy, regenerate, thumbs up/down near each response
5. **Model Badge**: Show active model name near input
6. **Token Counter**: Display estimated tokens used/remaining
7. **Sidebar**: Collapsible conversation history or quick settings
8. **Empty State**: Helpful starter prompts when no conversation exists

### Color Scheme Suggestions
**Light Mode:**
- Background: `bg-gray-50`
- Cards: `bg-white`
- Text: `text-gray-900`
- Accent: `bg-blue-600` for buttons

**Dark Mode:**
- Background: `bg-gray-950`
- Cards: `bg-gray-900`
- Text: `text-gray-100`
- Accent: `bg-blue-500` for buttons

### Animation Examples (Framer Motion)
```typescript
// Fade in messages
<motion.div
  initial={{ opacity: 0, y: 20 }}
```

## Example Feature: Streaming Text Effect

```typescript
const StreamingText = ({ text }: { text: string }) => {
  const [displayedText, setDisplayedText] = useState('');
  
  useEffect(() => {
    let index = 0;
    const interval = setInterval(() => {
      if (index < text.length) {
        setDisplayedText((prev) => prev + text[index]);
        index++;
      } else {
        clearInterval(interval);
      }
    }, 20); // Adjust speed here
    
    return () => clearInterval(interval);
  }, [text]);
  
  return <p>{displayedText}</p>;
};
```
