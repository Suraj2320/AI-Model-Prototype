# AI Interface Frontend Assessment - Complete Build Prompt

## Project Overview
Build a polished, frontend-only prototype of an AI chat interface (similar to ChatGPT, Claude, Gemini) that showcases modern UI/UX design and React/TypeScript development skills.

## Tech Stack (REQUIRED)
- **Framework**: Next.js OR React.js (functional components only)
- **Language**: TypeScript with strict mode enabled
- **Styling**: Tailwind CSS
- **Deployment**: Netlify, Vercel, or GitHub Pages

---

## Phase 1: Research & Planning

### AI Platforms to Review (Choose 3-5)
- OpenAI Playground
- Hugging Face Spaces
- Anthropic Claude UI
- Microsoft Copilot Lab
- Google Gemini
- Perplexity AI
- Any other AI chat interface

### Research Documentation (for README)
For each platform reviewed, write **1-2 sentences** summarizing:
- What makes their interface intuitive
- Key standout features you noticed
- Any unique interaction patterns

### Feature Selection
Choose **4-6 features** from your research to implement. Examples:
- Model/temperature selection
- Conversation history sidebar
- Copy/regenerate response buttons
- Syntax highlighting for code
- Streaming text effect
- Token counter
- Export conversation
- Prompt templates/examples

---

## Phase 2: Design (Figma/Adobe XD)

### Create Mockups for:
1. **Desktop view** (1440px width)
2. **Tablet view** (768px width)
3. **Mobile view** (375px width)

### Design System Using Tailwind
Document your choices:
- **Color Palette**: 
  - Light mode: background, text, accent colors
  - Dark mode: background, text, accent colors
- **Typography**: Font families, sizes (text-sm, text-base, text-lg, etc.)
- **Spacing**: Consistent padding/margin values (p-4, m-6, etc.)
- **Component Styles**: Buttons, inputs, cards, etc.

### Map Tailwind Tokens
For each mockup element, specify the exact Tailwind classes you'll use:
- Example: "Header uses `bg-gray-900 text-white p-4 shadow-lg`"

### Design Section in README
Include a "Design" section explaining:
- Your design decisions and why
- How Tailwind tokens map to mockup elements
- Screenshots or links to Figma/XD files
